# ⌨️ Keybinds

Labwc uses standard Openbox-style XML keybindings. The main modifier is `W` (Super/Windows key).

## 🖥️ Programs & System

| Keybind | Action |
| :--- | :--- |
| `W + Return` | Open Terminal (`lxterminal`) |
| `W + Q` | Close focused window |
| `W + Up` | Maximize/Restore window |
| `W + Down` | Minimize (Iconify) window |
| `W + Print` | Take screenshot (Custom script) |
| `A + F4` | Close window (Classic binding) |

## 🔍 Focus & Window Switching

| Keybind | Action |
| :--- | :--- |
| `A + Tab` | Next Window |
| `A + Shift + Tab` | Previous Window |

## 📐 Snapping & Movement

| Keybind | Action |
| :--- | :--- |
| `W + Left` | Snap to Left Half |
| `W + Right` | Snap to Right Half |
| `W + Up` | Snap to Top Half / Maximize |
| `W + Down` | Snap to Bottom Half / Minimize |
| `W + A` | Toggle Maximize |

## 🖱️ Mouse Bindings

| Action | Binding |
| :--- | :--- |
| `W + Left Click Drag` | Move Window |
| `W + Right Click Drag` | Resize Window |
| `Desktop Right Click` | Open Root Menu |
| `Titlebar Right Click` | Open Client Menu |

## 🔊 Media Keys

| Keybind | Action |
| :--- | :--- |
| `Volume Up/Down` | Master Volume 5% +/- (`amixer`) |
| `Mute` | Toggle Master Mute |
| `Brightness Up/Down` | Backlight 10% +/- (`brightnessctl`) |
