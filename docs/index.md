# 🏠 Welcome to my Labwc Config

A lightweight, Openbox-style stacking Wayland compositor setup on Arch Linux. This configuration embraces the classic floating paradigm with powerful edge-snapping, a classic Clearlooks theme, and a highly customized Waybar featuring pop-out drawers for power and battery options.

![Labwc Showcase](screenshots/labwc_screenshot_2026-05-24_17-40-09.jpg)

## ✨ Highlights
- **Classic Aesthetics:** Uses the Clearlooks-3.4 theme with 8px rounded corners.
- **Waybar Drawers:** Grouped power and battery modules that expand on hover for a clean UI.
- **Theme Switcher:** Click the backlight icon in Waybar to switch themes on the fly.
- **Openbox Familiarity:** XML configuration (`rc.xml`, `menu.xml`) with standard keyboard/mouse bindings.
