# labwc Config for Arch Linux

Personal labwc compositor config with Waybar, weather, and theme switcher.

## 📁 Structure
~/.dotfiles/labwc/
└── configs/labwc/
    ├── rc.xml          # Main config
    ├── waybar/         # Waybar config + weather scripts
    └── scripts/        # Helper scripts
